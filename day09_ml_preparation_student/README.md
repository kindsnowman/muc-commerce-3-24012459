## 用自己的话回答

- 什么是特征，什么是标签：特征是用于预测的输入变量，如用户年龄、消费金额等，标签是待预测的目标变量，即用户是否流失
- 为什么要保留测试集：只有测试集上的性能才能客观衡量模型是否真正学到了可迁移的规律，而非过拟合了特定数据子集
- 为什么83%准确率仍可能没有用：若数据中83%的用户本就未流失，模型只需将所有样本预测为“不流失”，即可轻松达到83%准确率，却完全无法识别出真正需要干预的流失用户，业务价值为零。
